# Pasos para configurar proyecto

- Instalar Python
- Crear entorno virtual(dentro de la carpeta del proyecto) ```python -m venv env```
- ```cd env/Script/activate```
- ```cd ..```
- ```pip install -r requirements.txt```
- ```python app.py```
